TfL journey planner
design improvements

portfolio project by Roberta Silva

Problem definition

The current journey planner page of Transport for London website has all the results beneath the fold when viewed on mobile.

My assumption is that it is difficult for users to discover the available travel options.

My design process

I’ve created a process to have a solid foundation for my work as a designer within different organisations.

I’m following the same flow when I’m walking through my case studies in this portfolio.

Improved version

I’ve mocked up a variant where I have merged duplicate Journey Results heading within the breadcrumb menu and moved Travel & Accessibility Options control next to Edit button – hence making space for a journey result.

I’ll be carrying out A/B testing to see which variant performs better:
